# Khmelevsky Furniture Studio — сайт-визитка

Одностраничный сайт мебельной студии на **React + Vite + Tailwind CSS**.
Чёрно-белый минималистичный дизайн, адаптив под телефоны и десктоп.

## Что на сайте

- **Главный экран (Hero)** — логотип, услуги бегущей строкой, заголовок, кнопки «Обсудить проект» (Telegram) и телефон.
- **Клиенты** — бегущая строка с брендами.
- **Работы** — галерея проектов с фильтром по категориям.
- **Направления** — 6 услуг студии.
- **О студии** — текст + преимущества.
- **Контакты** — Telegram, телефон, почта, город.
- **Подвал** — навигация и контакты.

## Как запустить (на компьютере с рабочим Node.js)

> Нужен установленный **Node.js 18+**. Проверьте: `node -v`.

```bash
npm install      # установка зависимостей (один раз)
npm run dev      # запуск в режиме разработки → откройте http://localhost:5173
```

Для готовой версии сайта (файлы для хостинга):

```bash
npm run build    # собирает папку dist/
npm run preview  # локальный просмотр собранной версии
```

Содержимое папки **`dist/`** после сборки — это и есть готовый сайт,
который можно загрузить на любой хостинг (Reg.ru, Timeweb, Vercel, Netlify и т.д.).

## Как добавить свои фотографии работ

Откройте папку `public/images/portfolio/` и прочитайте файл
`_КАК-ДОБАВИТЬ-ФОТО.txt`. Коротко: скопируйте фото в эту папку с нужными
именами — они появятся на сайте автоматически, без правок кода.

## Публикация на GitHub Pages (без установки npm)

GitHub сам соберёт сайт в облаке — программы на вашем компьютере не нужны.
В проекте уже лежит файл `.github/workflows/deploy.yml`, который всё делает автоматически.

**Шаг 1. Аккаунт и репозиторий**
1. Зарегистрируйтесь на https://github.com (если ещё нет аккаунта).
2. Нажмите «+» в правом верхнем углу → **New repository**.
3. Имя репозитория, например: `khmelevskiy-furniture-studio`.
4. Тип — **Public**. Ничего больше не отмечайте. **Create repository**.

**Шаг 2. Загрузить файлы проекта**
1. На странице репозитория: **Add file → Upload files**.
2. Перетащите в окно **всё содержимое** папки проекта
   (`index.html`, `package.json`, папки `src`, `public`, `.github` и остальные).
   Папку `node_modules` загружать не нужно (её и нет).
3. Внизу нажмите **Commit changes**.

> Если папка `.github` не загрузилась через перетаскивание (браузеры иногда
> прячут папки с точкой) — создайте файл вручную: **Add file → Create new file**,
> в поле имени введите `.github/workflows/deploy.yml`, вставьте его содержимое,
> **Commit**.

**Шаг 3. Включить GitHub Pages**
1. В репозитории: **Settings → Pages**.
2. В разделе **Build and deployment → Source** выберите **GitHub Actions**.

**Шаг 4. Дождаться сборки**
1. Откройте вкладку **Actions** — там пойдёт задача «Deploy to GitHub Pages».
2. Через 1–2 минуты появится зелёная галочка.
3. Адрес сайта: `https://ВАШ-ЛОГИН.github.io/khmelevskiy-furniture-studio/`
   (также он показан в **Settings → Pages**).

**Обновление сайта / добавление фото**
Загрузите новые файлы (например, фото в `public/images/portfolio/`) через
**Add file → Upload files** — сайт пересоберётся сам за пару минут.

**Свой домен (по желанию):** **Settings → Pages → Custom domain**.

## Где менять контакты и тексты

| Что | Файл |
|-----|------|
| Телефон, почта, Telegram, город | `src/App.jsx` → блок `CONTACTS` |
| Список и подписи работ | `src/data/portfolio.js` |
| Тексты «О студии» и «Направления» | `src/App.jsx` |
| Цвета, отступы, оформление | `src/App.css` |
| Логотип, иконка сайта | `public/logo.png` |

## Контакты студии

Все контакты (телефон, Telegram `@khmelevskyfurniture_bot`, почта
`info@m-style.pro`, город Ростов-на-Дону) хранятся в блоке `CONTACTS`
в файле `src/App.jsx` — там же их можно изменить.

### Важное примечание (среда генерации)

Сайт создан на рабочем компьютере, где:
- **Kaspersky Endpoint Security** блокирует загрузку файлов с Яндекс.Диска —
  поэтому фотографии работ подгрузить автоматически не удалось (добавьте вручную, см. выше);
- **npm повреждён** (нет модулей `@npmcli/config`, `graceful-fs`), поэтому проект
  не был собран и запущен здесь. Запускайте на другом ПК либо переустановите Node.js.

Технически проект полностью готов — нужно только выполнить `npm install`.
