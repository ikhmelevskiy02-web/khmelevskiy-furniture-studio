# Фотографии портфолио

Положите сюда изображения для сайта и пропишите их в `../photos.config.js`.

Пример:

```js
window.KFS_PHOTOS = [
  "./photos/kitchen-1.jpg",
  "./photos/wardrobe-1.jpg",
  "./photos/closet-1.jpg",
];
```

Рекомендации:

- формат: JPG или WebP;
- ширина: 1600–2400 px;
- вес файла: до 500–800 КБ после сжатия;
- имена файлов лучше писать латиницей без пробелов.
